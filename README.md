# findy-scratch-cs50

This is a Scratch game I built for CS50.

## What it does
The game changes scenes in a loop using a custom block I created. It includes multiple sprites, variables, and basic game flow.

## What I learned
- how to use loops and conditionals  
- how to create and use custom blocks with inputs  
- how to organize a simple interactive project
